Solubility of a System in Row-Echelon Form

Tuesday, 15 May 2018

1:43 PM

  • If the right-hand column is a leading column - no solution

  • If the right-hand column is not a leading column - at least one solution

 

  • If every variable is a non-zero value - one solution
  • If there is at least one non-leading variable - infinite solutions

 

  • If there are no non-leading rows - at least one solution
  • If there are no non-leading columns - at most one solution
  • If there are no non-leading columns nor non-leading rows - one solution
